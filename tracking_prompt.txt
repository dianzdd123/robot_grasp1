### **简化状态机设计**

```
IDLE → SEARCHING → TRACKING → APPROACHING → GRASPING → COMPLETED
```

**状态定义：**
- **IDLE**: 等待状态
- **SEARCHING**: 寻找目标 
- **TRACKING**: z=350mm水平追踪，优化xy+yaw对齐
- **APPROACHING**: 渐进下降z=350→270mm，持续追踪调整
- **GRASPING**: z=270mm最终抓取，微调xy+yaw
- **COMPLETED**: 抓取完成

### **高度控制策略**

```python
height_phases = {

    'GRASPING': 270         # 抓取高度

alignment_thresholds = {
    'xy_tolerance': 10.0,      # mm - xy对齐容差
    'yaw_tolerance': 5.0,      # 度 - yaw对齐容差
    'descent_step': 5.0,       # mm - 每步下降距离
}
```

**控制逻辑：**
1. **TRACKING阶段**：xy误差<10mm且yaw误差<5°持续2秒 → 转入APPROACHING
2. **APPROACHING阶段**：以5mm/步下降，对齐丢失暂停下降重新对齐
3. **GRASPING阶段**：到达z=270mm，最终微调后闭合夹爪

### **实时检测与ID匹配流程**

**⭐⭐⭐ 核心检测流程：**
1. **订阅实时图像** → `/camera/color/image_raw` + `/camera/depth/image_raw`
2. **YOLO检测** → 获取所有对象bbox + class_id
3. **SAM2分割** → 在每个bbox内提取精确mask  
4. **特征提取** → 从mask计算hu_moments + color_histogram
5. **ID匹配** → 与参考特征数据库对比匹配
6. **追踪控制** → 生成控制命令并执行

**ID匹配算法：**
```python
# 三层匹配机制

2. hu_moments相似度 (SAM2 mask计算)  
3. color_histogram相似度 (SAM2 mask提取)
4. 空间连续性奖励 (位置预测)

similarity = 0.4*hu_similarity + 0.4*hist_similarity + 0.2*spatial_bonus
```

### **yaw角度计算**

**使用最小外接矩形方法：**
```python
def _calculate_optimal_yaw(self, detection):
    rect = cv2.minAreaRect(largest_contour)
    _, (width, height), angle = rect
    
    # ⭐⭐⭐ 您的yaw计算公式核心 ⭐⭐⭐
    if width > height:
        yaw_angle = 90 - angle     # 长边方向
    else:
        yaw_angle = -angle         # 短边方向
    
    # 与当前yaw对比，选择最小转变
    yaw_options = [yaw_angle, yaw_angle + 180, yaw_angle - 180]
    best_yaw = min(normalized_options, key=lambda x: abs(x - current_yaw))
```

```python
def _collect_scan_info(self):
    """收集扫描信息用于tracking回退策略"""
    scan_info = {'center_pose': None, 'bounds': None, 'waypoint_poses': []}
    
    if self.fusion_mapping_data:
        waypoint_contributions = self.fusion_mapping_data.get('waypoint_contributions', {})
        poses = []
        for wp_data in waypoint_contributions.values():
            waypoint_info = wp_data.get('waypoint_data', {})
            world_pos = waypoint_info.get('world_pos')
            if world_pos:
                poses.append({
                    'position': {'x': world_pos[0], 'y': world_pos[1], 'z': world_pos[2]},
                    'orientation': {
                        'roll': waypoint_info.get('roll', 0),
                        'pitch': waypoint_info.get('pitch', 0), 
                        'yaw': waypoint_info.get('yaw', 0)
                    }
                })
        
        scan_info['waypoint_poses'] = poses
        
        # 计算扫描中心
        if poses:
            center_x = sum(p['position']['x'] for p in poses) / len(poses)
            center_y = sum(p['position']['y'] for p in poses) / len(poses)
            scan_info['center_pose'] = {
                'position': {'x': center_x, 'y': center_y, 'z': 350},
                'orientation': {'roll': 179, 'pitch': 0, 'yaw': 0}
            }
    
    return scan_info
```

```python
def _manual_imgmsg_to_cv2(self, msg, encoding):
    """手动转换ROS图像消息避免cv_bridge冲突"""
    image_array = np.frombuffer(msg.data, dtype=np.uint8/np.uint16)
```
